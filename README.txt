Стъпки за подкарване на проекта:

1. Свалете node.js от http://nodejs.org/ и го инсталирайте на желано от вас място
2. Инсталирайте модула Еxpress глобално с командата npm install -g express
3. През конзолата отидете в главната поддиректория на todoapp на вашия компютър
4. Изпълнете командата npm install (тя ще ви инсталира всички необходими модули, които са зададени във файла package.json, и ще ги сложи в папка node_modules)
5. За да стартирате сървъра изпълнете командата node app.js
6. Сървърът ще е достъпен на localhost:55555
